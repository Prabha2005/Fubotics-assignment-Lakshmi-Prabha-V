# Fubotics Chat Interface (Frontend + Backend)

A simple chat application built for the Fubotics internship task using Node.js (Express) backend and React frontend.  
Features message saving, CORS support, and environment-based configuration.

# backend .env
PORT=4000
FRONTEND_URL=http://localhost:3000

# frontend .env
REACT_APP_BACKEND_URL=http://localhost:4000
